Validation Testing

This phase of testing involves validating the Health and Fitness Tracking application against the functional and non-functional requirements elicited in the design. The tests listed below consider the application as a whole in terms of functionality, while also modulating test cases based on the different services in place to facilitate the modules. Most of the requirements presented in the design, are testable, with the exception of a few which would need further clarity, based on the target user interface design and market feedback. The requirements surrounding NFR-4, NFR-9, and NFR-5, were not directly testable.

Test Case ID Description Steps to Test Expected Results Requirements Validated
TC-001 Verify that a new user can register using email, phone number, or social media accounts. 1. Navigate to the registration page.
2. Enter password
3. Submit Form 		The user receives a confirmation message, and a new profile is created. FR-1
TC-002 Ensure users can log in securely using valid credentials. 1. Navigate to the login page.
2. Enter valid email and password.
3. Attempt to log in. 		User is authenticated and redirected to the dashboard. FR-1
TC-003 Verify the functionality of 2FA for enhanced security. 1. Enable 2FA in user settings.
2. Log out and attempt to log in again.
3. Enter the verification code sent via SMS or email. 		User can only log in after providing the correct 2FA code. FR-1
TC-004 Verify users can update their profile information. 1. Navigate to the profile settings page.
2. Modify fitness goals and dietary preferences.
3. Save changes. 		Changes are saved, and updates are reflected in the profile. FR-3
TC-005 Ensure users can log workouts using the exercise library. 1. Navigate to the workout logging page.
2. Select an exercise from the library.
3. Enter duration and intensity details.
4. Save the workout. 		Workout details are logged and appear in history. FR-2
TC-006 Verify that users can create and save custom workout routines. 1. Open the custom workout creation feature.
2. Add exercises, set repetitions, and define intensity.
3. Save the routine. 		The routine is saved and accessible for future use. FR-5
TC-007 Verify progress visualization for workouts. 1.Navigate to the progress dashboard.
2. Select workout data for a specific date range.
3. View metrics such as calories burned and exercise duration. 		Graphs and charts display accurate workout analytics. FR-4
TC-008 Verify users can log meals using the food database. 1.Open the nutrition tracking module.
2. Search for a food item or manually input details.
3. Log meal data including calories and macronutrients. 		Meal data is saved and reflected in daily totals. FR-3
TC-009 Ensure users can update dietary preferences and calorie goals. 1.Navigate to dietary settings.
2. Modify calorie and macronutrient targets.
3. Save changes. 		Goals are updated and considered in future insights. FR-3, FR-5
TC-010 Verify insights and patterns based on logged meals. 1.Navigate to the analytics dashboard.
2. View diet trends over a specified time period.
3. Compare calorie intake with goals. 		Insights accurately reflect logged data and highlight discrepancies. FR-5, FR-4, FR-2, NFR-3
TC-011 Ensure reports accurately combine workout and nutrition data. 1.Open the progress tracking module.
2. Generate a report for a chosen date range.
3. Analyze workout and diet summaries. 		The report displays comprehensive and accurate data. FR-4, NFR-3
TC-012 Verify tailored feedback based on user activity. 1. Log workouts and meals for a week.
2. Access the feedback section.
3. View recommendations. 		Feedback aligns with logged data, offering actionable advice. FR-6, NFR-3
TC-013 Verify users can share achievements with the community. 1.Navigate to the social feed.
2. Create a post with an image and caption.
3. Share the post. 		The post is visible to other users and accepts likes/comments. FR-6
TC-014 Ensure users can join and participate in challenges. 1.Browse available challenges.
2. Join a challenge and log related activities.
3. Track performance in the leaderboard. 		Challenge participation and progress are accurately recorded. FR-6,FR-5
TC-015 Test system performance with 1,000+ concurrent users. 1.Simulate 1,000+ users logging in and performing activities.
2. Monitor response times and system behavior. 		No performance degradation; response times remain less than 2 seconds. FR-8, NFR-6, NFR-1
TC-016 Ensure all sensitive data is encrypted at rest and in transit. 1.Review system logs during data transmission.
2. Verify encryption protocols are active. 		All sensitive data is securely encrypted. FR-7, NFR-2, NFR-7
